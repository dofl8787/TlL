	-문제 조건에 해당하는 값이 max 혹은 min이 되는
     경우를 찾는문제

최적화 문제는 결정 문제로 바꿔서 풀 수 있고, 결정 문제도 최적화 문제로 바꿔서 풀 수 있음(사촌지간이다~)

왜 중요한가? (최적화 문제가 가장 어려운 문제 유형)
-> 가장 어렵다는 말? 시간이 오래 걸린다(답을 구하기 까지의 시간이)
-> 지수승 이상의 시간 복잡도를 요하는 문제
2^n ~ n! 정도 이상의 시간 복잡도 (2^n 부분 집합, n! 순열)
-> 아직까지 아무도 더 좋은 알고리즘을 발견하지 못햇다
준내 똑똑한 똑똑이들이 작성해도 저정도 걸린다 ㅎㅎ

==> 즉 최적해를 찾는 문제 : 모든 후보해(가능한 모든 경우)를 조사해야 한다.
