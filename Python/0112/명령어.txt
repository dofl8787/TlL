cd : change directory
-> cd- : 홈 디렉토리
-> cd/ :루트 디렉토리
-> cd.. -> 상위 디렉토리

mkdir : make directory
 -> 여러 개의 폴더를 생성하려면 공백으로 구분해서 작성

ls : 현재 작업중인 폴더에 포함된 파일, 폴더들의 목록을  출력
 -> -a : 숨김 파일을 포함한 모든 파일들을 출력
-> -l : 파일에 대한 추가적인 정보(ex 가장 최근에 수정된 날짜, 권한, 크기, 용량 등) 더 많은 정보를
보는 것은 -l 옵션을 주면 된다.

touch : 파일을 생성하는 명령어
-> 여러 개의 파일을 생성할 수 있습니다.
-> 숨김 파일 같은 경우에는 '.'으로 시작합니다.

rm : 파일 혹은 폴더를 삭제하는 명령어
->!! GUI(탐색창) 와(과) 달리, 삭제시 휴지통으로 가지 않습니다. !!
-> 삭제를 누르면 바로 삭제되기 때문에 조심하세여! 휴지통으로 가지 않습니다!!!!!
-> 디렉토리(폴더)를 삭제할 때에는 '-r'옵션을 부여합니다.
-> 파일명을 포현할 때 *사용하면 0개 이상의 어떤 문자열을 의미합니다

rmdir : 디렉토리 삭제 명령어(rm -r과 같음)

걍 닥치고 다 지울거야  => f 옵션 있긴 하지만 .. 나중에 알려줌

txt 파일만 지우고 싶다 = '*' 
-> rm *txt : txt로 끝나는 파일들을 모두 지우고 싶어요!
-> rm ssafy*txt -> ssafy로 시작해서 txt로 끝나는 모든 파일을 지우는 것.

start . => 탐색기가 열림
start . 파일이름  -> 폴더/파일을 여는 명령어


notepad sssafy_phone -> ssafy_phone 이름의 메모장이 열림


터미널 단축기
 화살표 위/아래 키 : 실행시킨 명령어들이 히스토리로 남아있어서 보임 (명령어 조회)
 tap : 폴더 / 파일 이름을 자동 완성 시켜줌
 tap 두 번 연속 치면 -> 중복되는 파일들 보여줌
 ctrl + e : 커서를 맨 뒤로,
 ctrl + a : 커서를 맨 앞으로
