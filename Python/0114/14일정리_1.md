```python
산술 연산자

5 - 3 = 2(뺄셈)
100 * 5 = 500(곱셈)
100 / 3 = 33.33(실수로 나옴)
100 // 3 = 33 (소수점 이하는 버려버림. 몫만 나옴)
100 % 3 = 1 (나머지를 계산해주는 연산자)
2 ** 5 = 2의 5승 (32)

--------------------------------------------------------

비교 연산자
 5 == 5 (같냐)
 5 != 5 (같지 않냐)
 3 >= 3 (같거나 크냐)

```

```
c나 자바 경우 -> 정수끼리 계산하면 반드시 정수가, 실수끼리 계산하면 반드시 실수가 나옴
```
